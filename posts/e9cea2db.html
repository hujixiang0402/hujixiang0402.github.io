对文章进行加密

发表于 2023-05-23|更新于2023-05-23|演示
|字数总计:1210|阅读时长:4分钟|阅读量:

前言

  • 你可能需要写一些私密的博客, 通过密码验证的方式让人不能随意浏览.
  • hexo-blog-encrypt插件就能很好地解决这个问题.

特性

  • 一旦你输入了正确的密码, 它将会被存储在本地浏览器的localStorage中. 按个按钮, 密码将会被清空. 若博客中有脚本, 它将会被正确地执行.
  • 支持按标签加密.
  • 所有的核心功能都是由原生的 API 所提供的. 在 Node.js中, 我们使用 Crypto. 在浏览器中, 我们使用 Web Crypto API.
